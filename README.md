# 🌐 My Portfolio Website

A clean, dark-themed personal portfolio built with pure HTML, CSS, and JavaScript.

## ✨ Features
- Responsive design (mobile + desktop)
- Smooth scroll & scroll-reveal animations
- Active nav highlighting on scroll
- Working contact form with success message
- Skills grid & project showcase

## 🗂️ File Structure
```
portfolio/
├── index.html   → Main page
├── style.css    → All styles
└── script.js    → Interactions
```

## 🚀 How to Run
Just open `index.html` in your browser. No build step needed.

## 🛠️ Customize
- Replace `Your Name` with your real name
- Add your real projects in the projects section
- Update social links (GitHub, LinkedIn, email)
- Replace `resume.pdf` with your actual CV

## 📁 Deploy to GitHub Pages
1. Push this folder to a GitHub repo
2. Go to **Settings → Pages**
3. Set source to `main` branch
4. Your site will be live at `https://yourusername.github.io/portfolio`
